# 🤖 ChatGPT Local en Streamlit (Español, UTF-8)

Este proyecto es una interfaz de chat local usando la API de ChatGPT (OpenAI) con Streamlit.

## 🧰 Requisitos

- Python 3.8 o superior
- Clave API de OpenAI

## 🚀 Cómo usar

1. Instala dependencias:

```bash
pip install -r requirements.txt
```

2. Crea un archivo `.env` con tu API Key:

```
OPENAI_API_KEY=tu_clave_api
```

3. Ejecuta la app:

```bash
streamlit run chat_ui.py
```

Accede en tu navegador a `http://localhost:8501`
